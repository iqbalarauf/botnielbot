Catatan Update BotOniel

30 Juli 2023, 10.06
SELAMAT PAGI DAN SALAM SEJAHTERA!
Sesuai janji dan komitmen saya untuk menyebarkan jokes  @C_OnielJKT48 demi mewujudkan kebahagiaan bersama, maka dengan ini saya umumkan project lanjutan setelah buku kumpulan humor, yaitu BotOniel!

BotOniel merupakan sebuah bot Telegram yang dibangun untuk kalian dapat mengakses kumpulan jokes Oniel secara random. Penggunaannya cukup dengan mengikuti video di atas atau baca instruksi /start

Kalian bisa akses di link http://t.me/botnielbot
Terima kasih

6 Agustus 2023, 13.05
[UPDATE 2.0]
Terdapat penambahan fitur yang dapat mempermudah teman-teman untuk mendapatkan jokes Oniel. Fitur command akan dijelaskan di thread selanjutnya. Input command '/jokes' dan '/quote' masih dapat digunakan seperti biasanya

[PENAMBAHAN FITUR]
Selain jokes yang berbentuk teks, saat ini teman-teman bisa mendapatkan jokes Oniel secara random dalam bentuk video (dapat diputar langsung dari Telegram mobile). Gunakan command '/video' untuk mendapatkan random jokes video Oniel!

[PENAMBAHAN FITUR]
Kalian pernah dengar jokes Oniel tapi bingung di acara apa? Tenang, kalian dapat mencari jokes Oniel cukup dengan satu kata kunci!. Gunakan command '/cari' + kata kunci untuk mendapatkan informasi mengenai jokes Oniel yang sedang dicari

[PENAMBAHAN FITUR]
Petunjuk penggunaan dan command yang dapat digunakan dapat diakses melalui '/help' atau dapat mengakses melalui menu yang tersedia di sebelah kiri input text (pojok kiri bawah)

Bagi teman-teman yang menemukan jokes Oniel yang belum ada, kalian bisa ikut berpartisipasi menambahkan jokesnya dengan menggunakan command '/input' Input menggunakan Google Form dan jokes yang masuk akan divalidasi terlebih dahulu. Updatenya bisa dicek melalui Spreadsheet

7 November 2023, 17.00 WIB
[Broadcast Message from BotOniel]
 
Selamat Sore. Kami informasikan mengenai pembaruan pada BotOniel. 
- Penambahan fitur baru, DAILY JOKES
Teman-teman bisa mendapatkan jokes Oniel secara rutin dan otomatis setiap hari dari BotOniel (kami jadwalkan pada pukul 20.00-21.00 WIB). Siapa tahu bisa jadi hiburan senggang teman-teman setelah seharian beraktivitas

Fitur "Daily Jokes" dapat diaktifkan dengan menggunakan command /getdailyjokes
Untuk mematikan fitur "Daily Jokes", silakan gunakan command /removedailyjokes

- Perbaikan bug pada Tweet yang tidak dapat menampilkan link preview
Jika masih mengalami error saat menampilkan preview link Twitter/X, teman-teman dapat mengaktifkan Link Preview pada pengaturan aplikasi Telegram
(Settings -> Privacy and Security -> Secret Chat -> Aktifkan button "Link Preview")
 
Dalam beberapa waktu ke depan, tidak akan ada pembaruan fitur kecuali jika ditemukan bug/error. Kami akan berfokus melengkapi database jokes Oniel yang kurang atau menambahkan yang belum ada. Teman-teman bisa berpartisipasi untuk input jokes dari Oniel dengan menggunakan command /input
 
Jika terdapat permasalahan pada bot ini, dapat melakukan mention ke akun X 'iqbalarpake48'
 
Terima kasih 
Iqbal AR
